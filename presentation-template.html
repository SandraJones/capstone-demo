Day$$Trader

Sandy Jones
July 9, 2016

Project Goals:

  • Create an app that allows user to get most recent closing price for a stock traded on the New York Stock Exchange.
  • Give user the ability to save favorite stocks' closing prices into a database for later recall.
  • Give user the ability to save notes of things to watch for a given stock.

Demo

Time for a live demo!

Lessons Learned:

  • When encountering problems, keep overall structure of app in mind.
  • Troubleshooting line by line is laborious, but very helpful.
  • Utilize all available human assistance and internet resources to get the job done.
